Antigravity 是什麼（簡要）

Antigravity 是 Google 的 AI 程式編輯器，和 Cursor 同屬「用 AI 輔助寫程式」的編輯器，但設計重點不同：

項目CursorAntigravity
基底以 VS Code 為基礎Google 自家 IDE
特色對話 + 補全、改碼，可接 MCP多個 AI 代理自主規劃、寫碼、測試、驗證
使用方式你下 prompt，AI 建議或改程式用自然語言描述目標，代理在編輯器／終端／瀏覽器裡執行
模型Claude、Gemini、GPT 等主打 Gemini，也支援 Claude、GPT
脈絡約 20 萬 token約 100 萬 token
推出方Cursor 公司Google
圖片與遊戲動畫：
Cursor 不能直接產生圖檔，只能幫你寫程式或提示詞去呼叫外部圖像 API。
Antigravity 可以在 IDE 內直接產圖：內建 Imagen 3、Nano Banana Pro（Gemini 3 Pro Image）等模型，用自然語言描述需求即可產圖並整合進專案（例如 UI mockup、插圖、系統圖）。
遊戲與動畫方面，Antigravity 可依你的描述讓代理寫出遊戲程式並搭配產圖，快速做出可玩遊戲與 UI／動態效果（例如幾分鐘內完成一個小遊戲的程式與視覺），適合 MVP 或 demo；是否可「直接產出動畫影片檔」依產品更新為準。

名詞說明

LLM
LLM 是 Large Language Model 的縮寫，中文常說「大型語言模型」。指用大量文字訓練、能理解與產生人類語言的 AI 模型；Claude、GPT、Gemini 都屬於 LLM。
Prompt（提示詞）
你輸入給 AI 的指令或問題。在 Cursor、Antigravity、ChatGPT、n8n 的 AI 節點等地方都會用到。AI 根據 prompt 產生回覆或執行動作。
Cursor
以 VS Code 為基礎的程式編輯器，內建 AI 助理。用 prompt 與 AI 對話，可選 Claude、Gemini、GPT 等模型，支援 MCP 與 Skill（技能） 擴充：MCP 連外部工具，Skill 提供可重用的流程與領域知識。與 Antigravity 同屬「AI 程式編輯器」類型。
Antigravity
Google 推出的 AI 程式編輯器（IDE），與 Cursor 類似，主打「代理優先」：用自然語言描述目標，由多個 AI 代理自主規劃、寫碼、測試與驗證。支援 Gemini、Claude、GPT；具多代理協作、約 100 萬 token 脈絡、內建瀏覽器自動化與沙盒環境；同樣支援 MCP 與 Skill（技能）。與 Cursor 的重要差異：可內建直接產圖（Imagen 3、Nano Banana Pro 等），並可讓代理寫遊戲程式＋產圖，快速做出可玩遊戲與 UI；Cursor 則不能直接產圖。有 Windows / macOS / Linux 桌面版。詳見上方「Antigravity 是什麼（簡要）」。
ChatGPT
OpenAI 的網頁版對話介面，使用 GPT 系列模型。適合一般問答、寫稿、翻譯；寫程式則常用 Cursor 等編輯器整合。
VS Code
微軟的程式碼編輯器，開放原始碼。Cursor 是基於 VS Code 的「加上了 AI」的版本，外觀與操作類似。
n8n
開源的工作流自動化工具（類似 Zapier、Make），用節點拖拉串接各種服務。內建或可安裝 OpenAI、Anthropic 等 AI 節點，可把 prompt 送給 Claude、GPT 等，並與 Gmail、Slack、資料庫、API 等串接，做自動化流程。
Claude
Anthropic 的 LLM，擅長對話、分析與程式。在 Cursor、n8n、或 Claude 官網都可使用。
Gemini
Google 的 AI 模型，具多模態與推理能力。在 Cursor、Google AI Studio、Bard 等產品中可選用。
GPT
OpenAI 的語言模型系列（如 GPT-4）。用於 ChatGPT、Cursor、n8n、或透過 API 整合到自己的應用。
Anthropic
開發 Claude 的公司，與 OpenAI、Google 同屬 AI 模型提供者。
OpenAI
開發 GPT 與 ChatGPT 的公司，提供 API 供 Cursor、n8n、自建服務等呼叫。
MCP（Model Context Protocol）
讓 AI 助理連到外部工具與資料的協定。Cursor 與 Antigravity 都支援 MCP，裝 MCP 伺服器後，AI 可呼叫資料庫、API、本機腳本等，擴充能力。Antigravity 另提供 MCP Store 與 Google Cloud（BigQuery、AlloyDB 等）預建整合。與 Skill 的關係：MCP 負責「連到哪」（手腳／工具），Skill 負責「怎麼做」（流程／方法）。
Skill（技能）
Cursor 與 Antigravity 都有。Skill 是可重用的指令集或知識包，教 AI 代理如何執行特定任務（例如部署到測試環境、程式碼審查、寫文件）。通常以 Markdown 或目錄形式存放（Cursor：.cursor/skills/、.agents/skills/；Antigravity：.agent/skills/、~/.gemini/antigravity/skills/）。可手動觸發（如 /skill-name）或由代理依任務語意自動套用。與 MCP 互補：Skill 提供「做法與流程」，MCP 提供「對外連線與工具」。

關聯摘要：你用 Prompt 在 Cursor、Antigravity、ChatGPT 或 n8n 下指令 → 各介面把 prompt 送給 Claude、Gemini、GPT 等模型。Cursor 與 Antigravity 都支援 MCP（連外部工具）與 Skill（擴充代理流程與領域知識）；n8n 可把 AI 與其他服務串成自動化流程；VS Code 是 Cursor 的基底編輯器。
