Antigravity 入門使用手冊

Google 的 AI 程式編輯器 · 代理優先、內建產圖、多代理協作

一、什麼是 Antigravity？

Antigravity 是 Google 推出的 AI 程式編輯器（IDE），與 Cursor 同屬「用 AI 輔助寫程式」的產品。最大差異是代理優先：你用自然語言描述目標，多個 AI 代理會在編輯器、終端、瀏覽器裡自主規劃、寫碼、測試與驗證。另具內建產圖（Imagen 3、Nano Banana Pro 等）、約 100 萬 token 脈絡、內建瀏覽器自動化與沙盒環境，並支援 MCP 連接外部工具。

二、系統需求

  • 作業系統：Windows（x64 / ARM64）、macOS（Apple Silicon 或 Intel）、Linux（如 Ubuntu 20.04+、Fedora、Arch）
  • 硬體：建議足夠記憶體與磁碟空間以跑多代理與產圖
  • 網路：需連線以使用 AI 與產圖功能

三、下載與安裝

  1. 前往 antigravity.google/download 或 antigravity.codes 下載頁。
  2. Windows：選擇「Download for x64」（或 ARM64），下載後執行安裝程式。
  3. macOS：依晶片選擇 Apple Silicon 或 Intel 版本，下載並安裝。
  4. Linux：選擇對應發行版套件或通用格式，依說明安裝。
  5. 安裝完成後啟動 Antigravity，依設定精靈完成初次設定。

四、首次啟動與設定精靈

第一次開啟會引導你完成以下步驟：

4.1 設定流程

  • Start Fresh：全新開始，建議新使用者選此項。
  • Import from VS Code / Cursor：從 VS Code 或 Cursor 匯入既有設定與擴充。

4.2 編輯器主題

選擇喜好的主題（如 Dark、Tokyo Night、Light），之後可在設定中更改。

4.3 AI 代理政策

決定代理如何執行與審核：

  • 終端執行：Turbo（自動執行）/ Auto（代理決定）/ Off（需你手動確認）。
  • 審核政策：Request Review（總是先審核）/ Agent Decides（代理決定）/ Always Proceed（直接執行）。

入門可選「Review-driven」或「Agent-assisted」，先看代理建議再決定是否套用。

4.4 鍵盤與擴充

選擇鍵盤綁定（Normal 或 Vim 模式），並可安裝推薦的擴充功能。

4.5 登入

使用 Google 帳戶（Gmail） 登入以啟用 AI 與產圖功能，並同意使用條款。

五、介面簡介

介面以「代理」為中心，與傳統純聊天不同：

  • 代理面板／Manager View：查看與管理多個 AI 代理、任務狀態、執行結果。
  • 編輯區：程式碼編輯，代理會直接在此寫入或修改檔案。
  • 終端：代理可在此執行指令（依你的執行政策）。
  • 瀏覽器：內建瀏覽器自動化，代理可預覽或操作網頁。
  • Artifacts：產出物（程式、mockup、產圖、錄影等）會打包成 Artifacts，可在此檢視與回饋（例如「按鈕改大一點」「主色改藍」）。

六、基本操作

6.1 用自然語言下任務

在代理的輸入處用一般句子描述你要的結果，例如：

  • 「做一個簡單的待辦列表網頁，要有新增與刪除」
  • 「幫這個按鈕產一張圖，風格要扁平、藍色系」
  • 「寫一個小遊戲：用葉子射擊，有物理效果」

代理會規劃步驟、寫碼、必要時產圖，並在編輯器與終端中執行。

6.2 多代理協作

可同時啟用多個代理（例如前端代理、全端代理），分工處理不同任務，在 Manager View 中查看進度與結果。

6.3 產圖與 Artifacts

當任務需要圖片時，代理會使用內建模型（Imagen 3、Nano Banana Pro 等）產圖並整合進專案。產出會出現在 Artifacts 中，你可直接對 Artifact 給回饋（如「按鈕大一點」「主色改藍」），代理會一併改圖與程式。

七、MCP 與進階整合

Antigravity 支援 MCP（Model Context Protocol），與 Cursor 一樣可連接外部工具與資料。內建 MCP Store，可從代理面板搜尋並安裝 MCP 伺服器；另提供與 Google Cloud（如 BigQuery、AlloyDB、Cloud SQL）的預建整合，設定方式多為 UI 或 mcp_config.json。

八、與 Cursor 的差異（簡要）

項目CursorAntigravity
基底VS CodeGoogle 自家 IDE
使用方式你下 prompt，AI 建議或改碼描述目標，多代理自主執行
內建產圖無有（Imagen 3、Nano Banana Pro 等）
MCP支援支援（含 MCP Store、Google Cloud）
登入GitHub / Google / 電郵Google 帳戶

九、注意事項

  • 隱私與條款：使用 Google 帳戶與 AI／產圖服務時，請留意 Google 與 Antigravity 的隱私與使用條款。
  • 執行政策：代理可在終端與環境中執行指令，建議入門時使用「需審核」或「代理決定」，避免誤執行。
