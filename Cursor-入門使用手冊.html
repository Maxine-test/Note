Cursor 入門使用手冊

以 VS Code 為基礎的 AI 程式編輯器 · 用自然語言寫程式、除錯、重構

一、什麼是 Cursor？

Cursor 是以 VS Code 為基礎的程式編輯器，內建 AI 助理。你可以用自然語言（Prompt）描述需求，AI 會幫你產生或修改程式碼、解釋程式、除錯、重構。支援多種模型（Claude、Gemini、GPT 等），並可透過 MCP 連接資料庫、API、本機腳本等外部工具。

二、系統需求

  • 作業系統：Windows 10/11 (64 位元)、macOS 10.15+、Linux（如 Ubuntu 18.04+）
  • 硬體：建議 4 核以上 CPU、8GB 以上記憶體（大型專案建議 16GB+）、至少 2GB 可用磁碟空間
  • 網路：需連線以使用 AI 功能

三、下載與安裝

  1. 前往官網 cursor.sh 或 cursor.com。
  2. Windows：下載 .exe，雙擊執行安裝。
  3. macOS：下載 .dmg，將 Cursor 拖到「應用程式」資料夾。
  4. Linux：下載 .deb 或 .rpm / AppImage，依發行版安裝（例如 sudo dpkg -i cursor-*.deb）。

四、首次啟動與登入

  1. 開啟 Cursor，若有更新可先完成。
  2. 使用 GitHub、Google 或 電子郵件 註冊／登入帳號。
  3. 依畫面選擇方案（有免費額度，進階功能可能需訂閱）。
  4. 可選擇預設 AI 模型（例如 Claude、GPT-4），之後可在設定中更換。

五、介面簡介

介面與 VS Code 類似，主要區塊：

  • 左側：檔案總管、搜尋、延伸模組等。
  • 中間：編輯區，可開多個分頁。
  • 右側或下方：AI 聊天面板（Chat），與 AI 對話、下指令。
  • 命令面板：Ctrl+Shift+P（Windows/Linux）或 Cmd+Shift+P（Mac）可叫出所有指令。

六、基本操作

6.1 開啟專案

選單 檔案 → 開啟資料夾（或 Ctrl+K Ctrl+O），選擇你的專案資料夾。AI 會依專案內容提供更準確的建議。

6.2 開啟 AI 聊天（Chat）

按 Ctrl+L（Mac：Cmd+L）開啟 AI 聊天面板。在輸入框打字描述需求，例如：

  • 「幫我寫一個讀取 CSV 的 Python 函式」
  • 「解釋這段程式在做什麼」
  • 「幫我重構這個函式，讓它更簡短」

AI 會回覆並可依你同意將程式碼插入或套用到檔案。

6.3 行內編輯／生成（Cmd+K）

在編輯器中選取一段程式碼（或把游標放在要修改處），按 Ctrl+K（Mac：Cmd+K），輸入簡短指令（例如「加錯誤處理」「改成 async」），AI 會直接在那段程式碼上修改或生成。

6.4 使用 @ 引用上下文

在 Chat 輸入框輸入 @ 可帶入上下文，例如：

  • @檔名：引用某個檔案內容。
  • @Codebase：讓 AI 搜尋整個專案。
  • @Web：允許 AI 查網路（若已啟用）。
  • @Docs：引用文件。

有助於 AI 更精準理解你的專案與需求。

七、常用快捷鍵

Ctrl+L（Mac Cmd+L）→ 開啟 AI 聊天面板
Ctrl+K（Mac Cmd+K）→ 行內 AI 編輯／生成
Ctrl+Shift+P（Mac Cmd+Shift+P）→ 命令面板
Alt+I → 插入 AI 生成的程式碼（依版本可能不同）
Alt+] → 生成註解；Alt+[ → 解釋程式碼（依版本可能不同）

八、選擇 AI 模型與 MCP

在設定或 Chat 面板中可切換模型（如 Claude、GPT-4、Gemini 等），依需求與額度選擇。若需連接資料庫、API 或本機腳本，可安裝並設定 MCP（Model Context Protocol） 伺服器，讓 AI 在對話中直接使用這些工具。

九、注意事項

  • 隱私與資安：程式碼可能傳送至 AI 服務端，敏感專案請留意公司政策與 Cursor 隱私說明。
  • 不能直接產圖：Cursor 無法像 Antigravity 那樣內建產生圖片，需透過你寫的程式或外部 API 產圖。
